# StillOn brand

## Brand idea

**StillOn** is the personal agent outpost that stays with your computer. The user can leave an always-on Mac, Windows PC, or Linux machine behind and continue working with local Claude Code and Codex agents from a tablet, phone, or browser.

- Primary line: **Leave your computer. Keep your agents.**
- Supporting line: **You go. Your agents stay on.**
- Chinese line: **离开电脑，Agent 继续干活。**
- Product category: **Personal agent outpost**
- Core capability: **Stay Mode**

## Naming architecture

StillOn is the permanent product name. Working-dog names are release editions that communicate growing capability. The current release edition is **Husky**.

Never use a dog name in place of the StillOn product name. Write editions as “StillOn — Husky release” or “StillOn, Husky edition.”

## Mark and color

The mark is a continuous S-shaped connection ending in an illuminated status point. It represents a computer that remains reachable while its owner is away.

- Deep navy: `#08111F`
- Navy: `#14233D`
- Signal blue: `#60A5FA`
- Ice blue: `#C9F3FF`

Use [`assets/stillon-mark.svg`](../assets/stillon-mark.svg) as the source asset. Do not add dog imagery to the product mark; dog imagery belongs only to release storytelling.

## Voice

StillOn should sound calm, capable, and reassuring. Prefer concrete language about leaving the computer, keeping agents running, and reconnecting from another device. Avoid claiming that StillOn is a cloud execution service: the agents and data remain on the user's computer.
